LaunchModel
End-to-End MLOps Platform — interactive demo
Static preview · no backend needed

LaunchModel packages the complete ML model lifecycle — train → track → containerize → serve → monitor → autoscale → detect drift — in one production-ready Docker Compose stack. The reference model predicts whether a person's income exceeds $50K/year from census attributes, but the platform around it (FastAPI serving, Prometheus metrics, MLflow tracking, Helm + HPA, drift CronJob, CI/CD to GHCR) is fully model-agnostic.

Quick Start
git clone https://github.com/rayenx2/LaunchModel.git
cd LaunchModel
docker compose up -d

# Inference API
curl -s localhost:8015/predict \
  -H 'Content-Type: application/json' \
  -d '{"age":42,"education":"Bachelors","occupation":"Prof-specialty","hours_per_week":45}' | jq .

# MLflow UI
open http://localhost:5015
